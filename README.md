# FairPlay

A pitch practice app built with React. Configure your session, start the timer, and rehearse with a live webcam feed and phase-based structure.

## Tech Stack

- React 19, React Router 7, Zustand
- Vite, Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
